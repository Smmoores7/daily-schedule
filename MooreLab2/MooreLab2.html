Table Basics

How to define a table?
You define a table using the <table> </table> tags and then adding rows using the <tr> </tr> tags as well as adding data with the <td> </td> tags respectively.
Using <tr>
This tag is used to define the rows of your table. It will contain one or more <th> </th> tags and <td> </td> tags respectively.
Using <th>
This tag is used to define the header cells of your table. Its text will be bold and centered by default.
Using <td>
This tag is used to define the data cells of your table. Its text will be regular and left-aligned by default.
Using <thead>
This tag is used to group header content within your table. It's used in conjunction with <tbody> </tbody> tags and <tfoot> </tfoot> tags respectively. This tag must have at least one <tr> </tr> tags inside.
Using <tfoot>
This tag is used to group footer content within your table. It's used in conjunction with <thead> </thead> tags and <tbody> </tbody> tags respectively. This tag must have at least one <tr> </tr> tags inside like the tag above.
Using <caption>
This tag is used to define a table caption. It must be inserted right after the <table> </table> tags. Its text will be center-aligned above a table by default.
What is the difference between colspan and rowspan?
Colspan makes a cell span over columns (vertical merging of cells) while rowspan makes a cell span over rows (horizontal merging of cells).
What is the difference between cells spacing and padding?
Cell padding alters the space between the cell edges and the cell content while cell spacing alters the space between each cell.
How do you define table borders using CSS?
You would use the border property as your property under the tables selector followed by a numerical value like 1px and a thickness like solid.
How do you control border widths using CSS?
You would use the border-width property under the table selector followed by a value that determines the thickness using thin, medium, and thick.
